
-------------**-- 주의 --**-------------
Xcode에서 실행 시, Xcode의 키보드 언어 설정이 영어로 되어 있는지 확인 (상단 바 'A' 모양 확인)하고 실행(Run)해야 ‘s’, ‘q’와 같은 키보드 입력이 제대로 작동함. 


//----------실행환경----------// 
MacBook Pro (13-inch, 2019) / Mac OS Catalina (버전 10.15.5) / Xcode Version 11.3.1 (11C504)

//-------------------------English-------------------------//
Select level with a mouse.
There are 3 stars each to show past points.

After selecting level,
Click one of black dots to select water flow start point.
Try to reach purple destinations with water flow.

Press 's' key to start water flow.
Avoid red circles.
Press 'left, right' key to select line.
Press 'up, down' key to rotate selected line.

Reach all of purple destinations before time-out.
If success, there pops an button to: goto menu, retry, next level
And shows point with a star. (0, 0.5, 1, 1.5, 2, 2.5, 3)
If failure, there pops an button to: goto menu, retry.

Keep playing to unlock levels.
Tip: The steeper the slope, the faster the water flows.

Press 'q' key to quit program at any time.

//---------------------------한국어--------------------------//
마우스로 레벨을 선택한다.
각 레벨마다 3개의 별로 과거 점수를 보여준다.

레벨 선택 후,
검은색 점들 중 하나를 눌러 물이 나오는 점을 선택한다. 
보라색 사각형들(목적지)에 물 흐름이 닿도록 플레이한다.

키보드 's'를 눌러 물 흐름을 시작한다.
붉은색 원들은 피하도록 한다. (못 피하면 Game Over)
왼쪽, 오른쪽 방향 키를 눌러 물받침 라인을 선택한다.
위, 아래 방향 키를 눌러 선택된 라인을 회전시킨다.

주어진 시간이 끝나기 전에 모든 보라색 목적지들에 닿도록 한다.
성공하면 메뉴로 돌아가기, 재시도, 다음 레벨의 버튼이 나오며 별점 형식으로 점수를 보여준다. (0, 0.5, 1, 1.5, 2, 2.5, 3)
실패하면 메뉴로 돌아가기, 재시도 버튼이 나온다.

계속 플레이하여 새로운 레벨을 활성화한다. (비활성화되어 있는 레벨: 회색)
팁: 경사가 가파를수록, 물이 더 빨리 흐른다.

키보드 'q'를 눌러 언제든지 프로그램을 종료한다.
